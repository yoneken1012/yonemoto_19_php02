# ①課題名
ブックマークアプリ（CRUD対応）

## ②課題内容（どんな作品か）
- PHPとMySQLを用いて、書籍名・URL・コメントを登録し、登録済みデータを一覧表示できるブックマーク管理アプリ。
- 一覧画面から、登録済みデータの更新（編集）・削除ができる。

## ③アプリのデプロイURL
- http://yone-ken.sakura.ne.jp/yonemoto_19_php02/index.php

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID：なし
- PW：なし

## ⑤工夫した点・こだわった点
- SQLインジェクション対策として、プリペアドステートメント＋バインド変数を使用した。
- 一覧画面（select.php）から更新・削除まで完結できるようにし、最小構成でCRUDを実装した。
  
## ⑥難しかった点・次回トライしたいこと（又は機能）
- 更新・削除を実装するために、GET/POSTの分岐（`?id=` や `?act=del` など）の整理に苦労した。
- 次回は以下も改善したい。
  - UIの見やすさ改善（一覧のレイアウト調整、ボタン配置）
  - 入力バリデーション強化（未入力・形式チェック）
  - 削除処理をPOST化するなど、より安全な実装
